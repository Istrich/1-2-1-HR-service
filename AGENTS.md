# Роли агентов для HR 1-2-1 Web

Краткая шпаргалка для Cursor и распределения задач. Полный процесс: [docs/WORKFLOW.md](docs/WORKFLOW.md).

| Роль | Фокус |
|------|--------|
| **Аналитик** | Понять задачу, карту кода (`app.py`, `static/index.html`), риски |
| **Архитектор** | План в `docs/plans/YYYY-MM-DD-<slug>.md` |
| **Автор ТЗ** | `docs/specs/*-tz.md` по шаблону `docs/specs/TEMPLATE.md` |
| **Ревьюер ТЗ** | Полнота, тестируемость критериев приёмки |
| **Implementer** | Код строго по утверждённому ТЗ |
| **Guardian** | Секреты, ошибки, минимальный дифф, логирование |
| **Test Engineer** | Pytest, моки OpenAI/ffmpeg при интеграционных тестах |
| **QA** | Прогон тестов, регресс, смоук из README |
| **Контекст-кипер** | [context.md](context.md) после значимых изменений |
| **Документер** | README и пользовательская дока при смене поведения |

Контекст проекта без секретов: [context.md](context.md).
