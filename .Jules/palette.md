## 2024-05-24 - Missing ARIA labels on icon-only buttons\n**Learning:** Found multiple icon-only buttons (like close buttons `I.x` and audio play/pause `I.play`/`I.pause`) without `aria-label` or `title` attributes, which makes them inaccessible to screen readers.\n**Action:** When creating icon-only buttons using the global `I` object for SVG icons, always add descriptive `aria-label` and `title` attributes in Russian (e.g., `aria-label="Закрыть" title="Закрыть"`).
